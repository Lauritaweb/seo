# 🌐 Proyecto Web con Optimización SEO

## 📖 Descripción general
Este proyecto consiste en el desarrollo de un sitio web aplicando criterios fundamentales de SEO on-page, estructura semántica en HTML5 y buenas prácticas técnicas de desarrollo frontend.  
El objetivo principal fue construir una web optimizada tanto para usuarios como para motores de búsqueda, asegurando accesibilidad, rendimiento y correcta indexación.

---

## 🎯 Objetivos del proyecto
- Aplicar técnicas de SEO on-page en un sitio real
- Implementar una estructura semántica correcta en HTML5
- Desarrollar un sitio responsive adaptable a múltiples dispositivos
- Optimizar recursos (imágenes, textos, estructura)
- Simular configuraciones técnicas de SEO (robots.txt y sitemap.xml)
- Incorporar buenas prácticas profesionales de desarrollo web

---

## 🛠️ Tecnologías utilizadas
- HTML5
- CSS3
- Bootstrap

---

## 🔍 Implementación SEO

### Metaetiquetas
Se implementaron correctamente las siguientes metaetiquetas:
- **Meta Title**: optimizado con palabras clave relevantes
- **Meta Description**: redactada para mejorar el CTR en buscadores
- **Meta Robots**: configuración para permitir o restringir indexación según el caso

### Estructura de Headings
Se utilizó una jerarquía lógica de encabezados:
- Un único **H1** por página
- Uso ordenado de **H2, H3, H4** para estructurar el contenido
- Mejora de la legibilidad y SEO semántico

### SEO Semántico
Se aplicaron etiquetas HTML5 como:
- `<header>`
- `<nav>`
- `<main>`
- `<section>`
- `<article>`
- `<footer>`

Esto mejora la comprensión del contenido por parte de los motores de búsqueda.

### Imágenes optimizadas
- Uso de atributo **ALT** descriptivo en todas las imágenes
- Mejora de accesibilidad y posicionamiento en buscadores

### Keywords y contenido
- Selección estratégica de palabras clave
- Distribución equilibrada (densidad controlada)
- Inclusión natural dentro de los textos
